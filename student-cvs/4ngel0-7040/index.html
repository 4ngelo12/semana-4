Comandos y Alías que uso en Linux 🐚

Comandos Descripción
cd Permite cambiar de directorio
mkdir Sirve para crear una nueva carpeta o directorio
pwd Imprime la dirección del directorio actual
rm Permite eliminar archivos brindando una ruta
Ctrl + u Elimina una linea entera
Comandos Descripción
alias npms='npm start' Llama un npm para ejecutar comandos de node
alias cl='clear' Sirve para limpiar la terminal
alias ..='cd ..' Sirve para retroceder dentro de un directorio
alias gaa='git add .' Sirve agregar todos los archivos al staging area
alias gc='git commit -m' Permite crear un nuevo commit
